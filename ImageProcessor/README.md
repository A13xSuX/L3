# ImageProcessor

### Сервис поддерживает:

встроенные HTTP-методы:

– POST /upload — загрузка изображения на обработку;

– GET /image/{id} — получение обработанного изображения;

– DELETE /image/{id} — удаление изображения.

фоновую обработку изображений:

– ресайз

– генерация миниатюры

– добавление водяных знаков;

### Установка

```azure
заполнить .env для конфига
docker compose up -d
cd cmd
go run main.go
```

по адресу localhost:8080/ - будет доступен интерфейс
